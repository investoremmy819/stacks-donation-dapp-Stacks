# Stacks Donation DApp

A simple decentralized donation app on the Stacks blockchain.

## Features
- Send STX donations to a fixed recipient
- Connect Stacks wallet with @stacks/connect
- Demonstrates basic contract calls using @stacks/transactions

## Tech Stack
- Clarity
- JavaScript
- @stacks/connect
- @stacks/transactions

## Project Structure

contracts/
  donation.clar
frontend/
  index.html
  app.js

## Setup
1. Install Hiro Stacks Wallet
2. Deploy donation contract to Stacks Testnet
3. Open frontend/index.html
4. Connect wallet and send donation
